# decorator

- #### 实例方法

  ```python
  def fun(self, args):
    pass
  ```


- #### @classmethod 类方法

  ```python
  @classmethod
  def fun(cls, args):
    pass
  ```


- #### @staticmethod 静态方法
  ```python
  @staticmethod
  def fun(args):
    pass
  ```


- #### @property
  
  ```python
  @property
  def foo(self):
    return self._foo
  
  ```
