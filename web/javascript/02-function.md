## Function

1. #### 函数声明

   ```javascript
   (1) 声明 
   function funName(parameters) {
   	// ***
   }
   
   (2) 函数表达式
   var x = function(parameters) {
   	// ***
   };
   
   (3) 自调用
   (function () {
   	// ***
   })();
   
   (4) 箭头函数
   (parameters) => { 
   	// ***
   }
   	var  x = (a, b) => a*b
   ```

2. #### 构造函数

   ```javascript
   function Funname(params) {
   	// ***
   }
   let x = new Funname(param);
   
   // 构造函数不能被再次调用，不保存在任何地方，只能被创建和调用
   ```
