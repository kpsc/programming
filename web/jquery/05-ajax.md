## AJAX

- ajax 与服务器交换数据，在不重载全部页面的情况下，实现对部分网页的更新
- AJAX：Asynchronous JavaScript and XML

1. #### load

   ```javascript
   load() 从服务器加载数据，并把返回的数据放入被选元素中
   $(selector).load(URL[, data, callback])
   ```

   

2. #### get

   ```javascript
   $.get(URL[, callback])
   ```

   

3. #### post

   ```javascript
   $.post(URL[, data, callback])
   ```

   
   
4. #### ajax

   ```javascript
   执行异步 AJAX 请求
   ```
